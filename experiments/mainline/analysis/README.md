# Analysis Scripts

主線分析腳本依主題分成：

- `prefix/`: prefix signals、MSP、oracle gain、re-entry、task structure
- `trace/`: trace taxonomy
- `routing/`: routing simulation / policy analysis
- `embedding/`: embedding probe / cross-benchmark clustering
- `maintenance/`: result backfill / progress / repair 類工具

原則上這裡只讀取 `outputs/` 下既有資料，不直接產主結果。
